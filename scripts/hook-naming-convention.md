# 鉤子任務命名規範

## 標準格式

所有 cron/heartbeat 鉤子任務需在日誌開頭註明：

```
## 任務：[任務名稱] - [時間]
```

### 範例

```markdown
## 任務：資源自動調整 - 2026-03-05 08:02

### 執行結果
✅ 系統正常
```

### 舊格式（相容）
```markdown
# 資源自動調整記錄
## 2026-03-05 08:02
```

### 新格式（首選）
```markdown
## 任務：資源自動調整 - 2026-03-05 08:02
```

## 放置位置
- `memory/performance/` - 效能相關
- `memory/` - 一般任務
