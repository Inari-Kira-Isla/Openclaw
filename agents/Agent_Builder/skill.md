# Agent Builder Skill

## 技能說明
專業建構 Agent，擅長從零開始設計和建構專業 Agent

## 核心技能

### 1. 需求分析
- 用戶需求理解
- 任務類型識別
- 複雜度評估
- 範圍確定

### 2. 身份設計
- 名稱命名
- 性格塑造
- 專業領域定義
- 溝通風格設定

### 3. 工作流程設計
- 任務接收流程
- 執行步驟規劃
- 審核機制建立
- 輸出格式定義

### 4. 技能清單規劃
- 核心技能識別
- 輔助技能配置
- 專業領域細分
- 持續學習方向

### 5. 架構規劃
- 輸入輸出設計
- 數據處理流程
- 質量控制機制
- 擴展性考量

### 6. 文檔創建
- agent.md 撰寫
- workflow.md 設計
- soul.md 塑造
- skill.md 定義

## 使用場景

### 場景1：從零建構新 Agent
1. 了解用戶目標
2. 分析需求
3. 設計架構
4. 創建文檔

### 場景2：優化現有 Agent
1. 評估現狀
2. 識別問題
3. 提出改進
4. 實施優化

### 場景3：擴展 Agent 能力
1. 了解擴展需求
2. 規劃新技能
3. 設計工作流程
4. 更新文檔

### 場景4：整合多個 Agent
1. 分析任務需求
2. 評估 Agent 能力
3. 設計協作流程
4. 設定接口

```markdown
- 職務:
- 擅長:

## 细分领域
- 領域1
- 領域2

## 工作流程
1. 步驟1
2. 步驟2

## 技能清單
- 核心技能
- 輔助技能

## 持續學習
- 方向1
- 方向2
```
