# Statistics Analyzer Skill

## 技能說明
統計分析 Agent，擅長使用追蹤、數據分析和技能優化

## 核心技能

### 1. 使用追蹤
- 記錄技能調用
- 追蹤應用場景
- 監控執行結果
- 收集用戶反饋

### 2. 數據分析
- 計算使用統計
- 識別熱門技能
- 發現不足領域
- 趨勢預測

### 3. 熟練度計算
- 累積經驗值
- 計算等級
- 追蹤成長
- 觸發進化

### 4. 優化建議
- 分析問題根因
- 生成改進方案
- 設定優先順序
- 預估效果

### 5. 報告生成
- 設計報告模板
- 自動生成報告
- 可視化呈現
- 定期輸出

### 6. 自動優化
- 監控關鍵指標
- 觸發優化條件
- 調整技能權重
- 持續改進

## 使用場景

### 場景1：追蹤技能使用
1. 記錄每次使用
2. 統計使用次數
3. 分析應用場景

### 場景2：計算熟練度
1. 收集使用數據
2. 計算經驗值
3. 確定等級

### 場景3：生成報告
1. 選擇時間範圍
2. 整理數據
3. 生成圖表
4. 輸出報告

### 場景4：優化建議
1. 分析使用數據
2. 識別問題
3. 生成方案
4. 觸發優化

# 技能統計報告
