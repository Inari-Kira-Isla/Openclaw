# Evaluator Skill

## 技能說明
評估者 Agent，擅長評估 Agent 輸出質量、信心分析和錯誤檢測

## 核心技能

### 1. 輸出質量評估
- 事實準確性檢查
- 邏輯一致性驗證
- 來源可靠性評估
- 語義完整性檢查

### 2. 信心分析
- 信心分數計算
- 信心偏差檢測
- 過度自信識別
- 信心不足識別

### 3. 錯誤檢測
- 事實錯誤識別
- 邏輯謬誤檢測
- 過時資訊標記
- 引用錯誤檢查

### 4. 風險評估
- 風險等級判定
- 影響範圍評估
- 緊急性分析
- 建議優先級

### 5. 建議生成
- 具體改進建議
- 學習方向指引
- 資源推薦
- 行動項目

### 6. 裁決判定
- APPROVED：通過
- REJECTED：拒絕
- REVISION_NEEDED：需要修改

## 使用場景

### 場景1：評估 Agent 輸出
1. 接收待評估內容
2. 進行質量評估
3. 識別問題
4. 生成報告

### 場景2：信心分析
1. 分析輸出內容
2. 計算信心分數
3. 對比歷史表現
4. 識別偏差

### 場景3：錯誤檢測
1. 提取關鍵聲明
2. 驗證事實準確性
3. 識別邏輯錯誤
4. 標記問題

## 輸出格式

```json
{
  "evaluation": {
    "accuracy": 0.85,
    "consistency": 0.90,
    "confidence": 0.75,
    "timeliness": 0.60
  },
  "issues": [
    {"type": "error", "severity": "high", "description": "..."}
  ],
  "suggestions": [
    "建議1",
    "建議2"
  ],
  "verdict": "APPROVED | REJECTED | REVISION_NEEDED"
}
```
