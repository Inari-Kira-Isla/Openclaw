# Skill Creator Workflow

## 工作流程

### 第一階段：接收需求

#### 1.1 接收 Agent Builder 的輸出
- 獲取架構設計
- 理解任務目標
- 確認專業領域

#### 1.2 分析需求
- 識別核心功能
- 評估技能複雜度
- 確定技能範圍

### 第二階段：設計技能

#### 2.1 核心技能設計
- 識別主要能力
- 定義關鍵功能
- 提煉專業技能
- 設定技能等級

#### 2.2 輔助技能設計
- 列出支援技能
- 識別通用能力
- 規劃增值服務
- 設定擴展方向

#### 2.3 專業領域設計
- 主要領域定義
- 次要領域規劃
- 交叉領域識別
- 新興領域探索

### 第三階段：建立技能樹

#### 3.1 技能節點設計
- 創建技能節點
- 設定節點屬性
- 定義輸出格式
- 建立連接關係

#### 3.2 依賴關係規劃
- 識別前置技能
- 規劃學習順序
- 建立依賴鏈
- 優化路徑

#### 3.3 進化路徑設計
- 設計升級路線
- 設定進化條件
- 規劃成長方向
- 建立激勵機制

### 第四階段：熟練度系統

#### 4.1 等級定義
- 初級（Novice）：基礎應用
- 中級（Intermediate）：獨立操作
- 高級（Advanced）：複雜問題
- 專家（Expert）：創新解決
- 大師（Master）：領域引領

#### 4.2 升級條件
- 使用次數閾值
- 任務完成質量
- 學習新技能
- 貢獻社區

#### 4.3 成長機制
- 經驗值累積
- 技能解鎖
- 領域拓展
- 自動優化

### 第五階段：統計分析整合

#### 5.1 使用追蹤
- 記錄使用次數
- 追蹤應用場景
- 監控成功率
- 分析滿意度

#### 5.2 數據分析
- 熱門技能識別
- 不足領域發現
- 學習方向建議
- 優化方案生成

#### 5.3 自動優化
- 技能權重調整
- 推薦技能優先
- 進化觸發
- 持續改進

## 輸出格式

```
## 技能清單

### 核心技能
| 技能 | 等級 | 使用次數 |
|------|------|----------|

### 輔助技能
| 技能 | 等級 | 使用次數 |
|------|------|----------|

### 專業領域
| 領域 | 熟練度 |
|------|--------|
```
