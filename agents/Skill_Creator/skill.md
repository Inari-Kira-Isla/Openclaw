# Skill Creator Skill

## 技能說明
專業技能設計 Agent，擅長設計核心技能、輔助技能和成長系統

## 核心技能

### 1. 核心技能識別
- 分析任務需求
- 識別關鍵能力
- 提煉核心功能
- 設定優先級

### 2. 技能架構設計
- 建立技能框架
- 設計技能層次
- 規劃技能路線
- 設定依賴關係

### 3. 熟練度系統
- 設計等級體系
- 設定升級條件
- 建立成長機制
- 追蹤熟練度

### 4. 技能樹構建
- 創建技能節點
- 設計進化路徑
- 優化學習曲線
- 建立激勵系統

### 5. 數據分析
- 追蹤使用數據
- 分析熱門技能
- 識別不足領域
- 生成優化建議

### 6. 自動優化
- 調整技能權重
- 觸發技能進化
- 持續改進系統
- 適應新需求

## 使用場景

### 場景1：設計新技能
1. 理解需求
2. 設計核心技能
3. 規劃輔助技能
4. 建立技能樹

### 場景2：優化現有技能
1. 分析使用數據
2. 識別問題
3. 提出改進方案
4. 實施優化

### 場景3：擴展技能範圍
1. 評估需求
2. 設計新技能
3. 整合到技能樹
4. 設定進化條件

### 場景4：建立成長系統
1. 設計熟練度等級
2. 設定升級條件
3. 建立激勵機制
4. 追蹤成長數據

## 輸出格式

```markdown
# Skills for [Agent Name]

## Core Skills
| Skill | Level | Description |
|-------|-------|-------------|

## Auxiliary Skills
| Skill | Level | Description |
|-------|-------|-------------|

## Professional Domains
| Domain | Proficiency | Growth Path |

## Skill Tree
[Visual representation of skill tree]

## Growth System
[Growth mechanism and levels]
```

- 技能設計模板
- 熟練度計算器
- 技能樹視覺化
- 使用數據分析儀表板

- 技能設計趨勢
- 遊戲化設計
- 機器學習優化
- 用戶體驗
