# RAG 檢索 Skill

**觸發條件：** 用戶需要搜尋知識、歷史記錄、專案研究、技能學習、決策支援

## 功能

| 場景 | 觸發詞 | 功能 |
|------|--------|------|
| Project 研究 | 研究、項目分析、調研 | 搜尋歷史項目、相關案例 |
| Skill 學習 | 學習、技能、能力 | 匹配最佳實踐、學習路徑 |
| Research Study | 研究、文獻、趨勢 | 檢索知識關聯、摘要 |
| 決策支援 | 決策、建議、評估 | 數據驅動、風險評估 |

## 檢索流程

1. **解析查詢** - 識別意圖和關鍵詞
2. **向量匹配** - 搜尋相似內容
3. **排序返回** - 相關性排序
4. **結果整合** - 格式化成回覆

## 使用方式

```
用户: 查找 OpenClaw 最近的專案
→ 觸發: project-research
→ 搜尋關鍵詞: OpenClaw, 專案, 2026
→ 返回相關向量結果
```
