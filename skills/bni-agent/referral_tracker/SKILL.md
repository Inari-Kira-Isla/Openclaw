# Referral Tracker

## 轉介紹狀態

| 狀態 | 描述 |
|------|------|
| 剛收到 | 新收到的轉介紹 |
| 聯繫中 | 已聯繫，待進一步 |
| 報價中 | 已報價，等待回覆 |
| 成交 | 成功成交 |
| 未成交 | 結束未成交 |

## 追蹤記錄
```
轉介紹：會員A → 會員B
產品：網站設計
金額：$5000
日期：2024-01-15
狀態：報價中
```
