# Decay Monitor

## 衰減模型

### 記憶強度公式
```
Strength(t) = Strength(0) × e^(-λt)
```

### 衰減率分類
| 類型 | λ | 適用記憶 |
|------|---|----------|
| 快速衰減 | 0.1/天 | 短期工作記憶 |
| 中速衰減 | 0.02/天 | 對話上下文 |
| 慢速衰減 | 0.001/天 | 長期知識 |

## 觸發條件

### 1. 時間觸發
- 距離上次訪問超過 N 天
- 總記憶時間超過閾值

### 2. 訪問觸發
- 相關關鍵詞被查詢
- 上下文觸發召回

### 3. 重要性觸發
- 高價值記憶優先刷新
- 重要人物/事件優先

## 刷新策略

### 主動刷新
- 使用時自然強化
- 背景定時刷新
- 關聯記憶同步刷新

### 被動刷新
- 查詢時更新時間戳
- 引用時增強權重

## 監控報告

```json
{
  "total_memories": 1000,
  "healthy": 800,
  "decayed": 150,
  "critical": 50,
  "refreshed_today": 20,
  "recommendations": [
    "建議刷新 50 個臨界記憶",
    "可歸檔 30 個長期未用記憶"
  ]
}
```

## 閾值配置

```json
{
  "decay_thresholds": {
    "healthy": 0.7,    // > 70% 強度
    "decayed": 0.4,   // 40-70% 強度
    "critical": 0.2   // < 40% 強度
  },
  "refresh_interval": 86400  // 24小時
}
```
