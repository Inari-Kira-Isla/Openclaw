# Memory Organization

## 分類維度

### 1. 時間維度
- 短期記憶（< 1 週）
- 中期記憶（1 週 - 1 年）
- 長期記憶（> 1 年）

### 2. 類型維度
| 類型 | 範例 |
|------|------|
| 事實 | 台北是首都 |
| 事件 | 開會記錄 |
| 人物 | 聯絡人資訊 |
| 偏好 | 喜歡的事物 |
| 技能 | 會的能力 |

### 3. 來源維度
- 用戶主動提供
- 系統觀察學習
- 外部資料攝入

## 標籤體系

### 層次標籤
```
人物/
├── 家人/
│   ├── 父母
│   └── 兄弟姐妹
├── 朋友/
│   └── 摯友
└── 同事/
    └── 主管
```

### 語義標籤
- #重要
- #待處理
- #已完成
- #需確認
- #已過時

## 組織策略

### 1. 自動分類
- 根據內容關鍵詞
- 根據來源
- 根據使用頻率

### 2. 手動分類
- 用戶指定分類
- Agent 建議分類
- 定期審查調整

### 3. 動態重組
- 相似記憶聚合
- 關聯記憶鏈結
- 過時記憶歸檔

## 歸檔規則

| 條件 | 動作 |
|------|------|
| 超過 1 年未訪問 | 建議歸檔 |
| 被標記為過時 | 自動歸檔 |
| 包含敏感資訊 | 分離儲存 |
| 用戶要求刪除 | 完全刪除 |
