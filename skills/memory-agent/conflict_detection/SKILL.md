# Conflict Detection

## 衝突類型

### 事實衝突
```
記憶1: 蘋果是紅色
記憶2: 蘋果是綠色
→ 需要驗證或標記
```

### 時間衝突
```
記憶1: 2024年1月在台北
記憶2: 2024年1月在紐約
→ 需要時間上下文
```

### 語義衝突
```
記憶1: 他不喜歡狗
記憶2: 他養了一隻狗
→ 可能是謊言或成長
```

## 檢測流程

```
新資訊 → 相似性搜尋 → 計算相似度 → 判斷衝突 → 標記/解決
```

### 相似度閾值
| 閾值 | 判定 |
|------|------|
| > 0.95 | 高度相似，可能重複 |
| 0.8-0.95 | 相似，需人工確認 |
| < 0.8 | 不視為衝突 |

## 解決策略

### 1. 標記策略
- 標記為「待確認」
- 保留所有版本
- 等待人工裁決

### 2. 覆蓋策略
- 信任新資訊
- 標記舊資訊為「過時」
- 保留兩者供参考

### 3. 合併策略
- 找出共同點
- 保留差異觀點
- 建立複合記憶
