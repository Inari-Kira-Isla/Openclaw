# 工作前智慧檢索 (Pre-task Retrieval)

## 功能說明

在執行任務前，進行「意圖對齊檢索」：
- 向量搜尋 (語義相似)
- 關鍵字搜尋 (精確術語)
- 上下文排名

## 輸入

| 欄位 | 類型 | 說明 |
|------|------|------|
| task | string | 當前任務描述 |
| context | string | 任務背景 |

## 輸出

| 欄位 | 類型 | 說明 |
|------|------|------|
| relevant_notes | array | 相關筆記列表 |
| search_results | array | 搜尋結果 |
| expanded_query | string | 擴展後的查詢 |

## 工作流程

```
1. 接收任務描述
   ↓
2. 生成搜尋查詢
   - 提取關鍵字
   - 擴展語義範圍
   ↓
3. 執行混合搜尋
   - 向量搜尋 (ChromaDB)
   - 關鍵字搜尋
   ↓
4. 整合結果
   - 去除重複
   - 排序相關性
   ↓
5. 上下文排名
   - 讓 Agent 評估是否相關
   - 若無關，擴大搜尋半徑
   ↓
6. 輸出結果
```

## 搜尋策略

### 向量搜尋
- 使用 ChromaDB 語義搜尋
- 回傳 top 5 結果

### 關鍵字搜尋
- 精確匹配術語
- 例如：海膽批號、專有名詞
