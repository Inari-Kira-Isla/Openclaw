# Anomaly Detector

## 檢測方法

### 1. 統計方法
- Z-Score 異常值
- 移動平均偏差
- 季節性分析

### 2. 模式識別
- 正常行為基準偏離
- 罕見事件檢測
- 序列異常

### 3. 學習方法
- 历史数据学习
- 動態閾值調整

## 基線建立

### 正常行為基準
```json
{
  "login_times": {"mean": 9, "std": 2},
  "request_count": {"mean": 100, "std": 20},
  "error_rate": {"mean": 0.01, "std": 0.005}
}
```

### 異常閾值
- 超過 3 個標準差
- 短時間異常內多次
- 持續偏離基準

## 異常類型

| 類型 | 範例 |
|------|------|
| 行為異常 | 異常登入時間 |
| 流量異常 | 請求激增 |
| 效能異常 | 回應變慢 |
| 錯誤異常 | 錯誤率上升 |

## 處理流程

```
檢測 → 評估嚴重性 → 分析原因 → 觸發響應 → 記錄學習
```
