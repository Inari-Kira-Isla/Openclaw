# Access Log Monitor

## 監控維度

### 1. 存取模式
- 存取頻率
- 存取時間
- 存取位置

### 2. 異常檢測
- 異常登入
- 異常流量
- 異常行為

### 3. 合規性
- 誰存取了什麼
- 何時存取
- 存取結果

## 日誌分析

### 常見模式
```
正常: 均勻分布，白天為主
異常: 凌晨高峰、異常來源、頻率激增
```

### 異常檢測規則
```json
{
  "rules": [
    {"type": "failed_login", "threshold": 5, "window": "5m"},
    {"type": "unusual_time", "hours": [0,1,2,3,4]},
    {"type": "high_frequency", "threshold": "100 req/min"}
  ]
}
```

## 報告格式

```
📊 存取日誌報告 - 2024-01-15

總存取次數: 1,234
成功: 1,200 (97%)
失敗: 34 (3%)

⚠️ 異常:
- 失敗登入 5 次 (IP: xxx)
- 異常時間存取 2 次
```

## 報警觸發
- 連續失敗登入
- 異常來源存取
- 權限提升嘗試
