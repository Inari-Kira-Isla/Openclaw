# Drift Detection

## 漂移類型

### 1. 效能漂移
- 回應時間增加
- 錯誤率上升
- 資源使用增加

### 2. 行為漂移
- 輸出分布改變
- 觸發模式改變
- 用戶互動模式改變

### 3. 數據漂移
- 輸入特徵分布改變
- 標籤分布改變
- 關聯關係改變

## 檢測方法

### 統計檢測
```python
# Z-Score
z = (x - mean) / std
if abs(z) > 3:  # 漂移

# PSI (Population Stability Index)
psi = Σ((A% - E%) * ln(A% / E%))
if psi > 0.2:  # 顯著漂移
```

### 趨勢檢測
- 移動平均
- 指數平滑
- 異常點檢測

## 閾值設定

### 預設閾值
| 指標 | 警告 | 嚴重 |
|------|------|------|
| 錯誤率 | > 1% | > 5% |
| 回應時間 | +50% | +100% |
| 記憶使用 | > 70% | > 90% |

## 報警流程

```
檢測 → 評估 → 分類 → 通知 → 處理
```

### 報警級別
- INFO: 輕微偏離
- WARN: 需要關注
- ERROR: 影響功能
- CRITICAL: 系統異常
