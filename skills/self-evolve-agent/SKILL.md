# 自我進化訓練 Skill

**版本**: 2.0
**優化**: 加入錯誤處理與輸出格式

---

## 說明

執行 AI 趨勢自我進化訓練，搜尋最新資訊並存入知識庫。

---

## 參數

| 參數 | 類型 | 預設 | 說明 |
|------|------|------|------|
| topic | string | "AI trends" | 搜尋主題 |
| limit | number | 5 | 結果數量 |
| model | string | "sonnet-4.5" | 使用模型 |

---

## 輸出格式

```markdown
## 訓練結果

### 主題: {topic}

### 發現 ({count} 項)
| # | 標題 | 來源 | 價值 |
|---|------|------|------|
| 1 | {title} | {source} | {rating} |

### 行動建議
- {action1}
- {action2}

### 錯誤記錄
{errors}
```

---

## 錯誤處理

| 錯誤 | 處理方式 |
|------|----------|
| 搜尋失敗 | 回報「搜尋失敗: {error}」，使用備用來源 |
| API 逾時 | 回報「逾時，切換模型重試」 |
| 無結果 | 回報「無結果，嘗試其他關鍵字」 |
| 儲存失敗 | 回報「儲存失敗: {error}」 |

---

## 設計原則

1. **精準設計** - 避免過多雜訊
2. **明確定義** - 輸出格式
3. **容錯機制** - 處理錯誤情況

```
- "執行 AI 趨勢訓練"
- "執行本地 LLM 訓練，限制 3 項"
```
