# Conflict Resolution

## 衝突類型

### 1. 資源衝突
- 同一資源的競爭
- 優先級衝突

### 2. 規則衝突
- 規則相互矛盾
- 權限衝突

### 3. 決策衝突
- 多方意見不同
- 目標衝突

## 解決策略

### 1. 協商
- 找出共同點
- 讓步達成共識

### 2. 仲裁
- 根據規則裁決
- 第三方調解

### 3. 升級
- 提交上級決定
- 管理層介入

## 輸出格式
```json
{
  "conflict": "描述",
  "resolution": "解決方案",
  "parties_agreed": true,
  "next_steps": ["步驟"]
}
```
