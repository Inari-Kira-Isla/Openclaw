# Value Scorer

## 評分維度

### 1. 購買力
- 營收規模
- 預算金額

### 2. 購買意願
- 詢問頻率
- 回覆速度

### 3. 戰略價值
- 產業相關性
- 品牌曝光

## 評分模型

| 等級 | 分數 | 描述 |
|------|------|------|
| A | 80-100 | 高價值 |
| B | 60-79 | 中價值 |
| C | 40-59 | 低價值 |
| D | < 40 | 觀察 |

## 建議
- A 級：優先跟進
- B 級：定期維護
- C 級：自動化互動
- D 級：低優先
