# Question Analytics

## 分析維度

### 1. 問題分布
- 類型占比
- 熱門問題排名

### 2. 回答效果
- 回覆率
- 解決率
- 客戶滿意度

### 3. 趨勢變化
- 每週變化
- 季節性
