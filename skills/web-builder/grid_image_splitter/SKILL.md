# 9宮格圖切割工具

## 功能需求

| 功能 | 說明 |
|------|------|
| 圖片上傳 | 支援上傳九宮格圖片或任何拼圖 |
| 自動切割 | 自動分割成9張單圖 |
| 切割預覽 | 顯示切割後的預覽 |
| 單張下載 | 可選擇下載單張圖片 |
| 批次下載 | 一鍵下載全部圖片 |
| ZIP 下載 | 打包成ZIP壓縮包下載 |
| 畫質提醒 | 提醒可至即夢/豆包進行高清修復 |

## 技術實現

- HTML5 Canvas API
- JSZip 函式庫（ZIP生成）
- FileSaver.js（檔案下載）

## 使用流程

1. 上傳九宮格圖片
2. 系統自動切割成9張
3. 預覽切割結果
4. 選擇單張或批次下載
