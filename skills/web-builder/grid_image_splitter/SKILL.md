# 9宮格圖切割工具

## 功能說明

將九宮格圖片或任何拼圖自動分割成獨立的單張圖片，支援預覽、單張下載及批次 ZIP 打包下載。

## 工作流程

### 第一步：接收圖片
- 用戶上傳一張九宮格圖片或拼圖
- 驗證圖片格式（支援 PNG、JPG、WEBP）

### 第二步：自動切割
- 預設切割為 3x3 共 9 張圖片
- 根據圖片尺寸均等分割
- 若用戶指定其他格式（如 2x2、4x4），按需調整

### 第三步：預覽確認
- 顯示切割後的所有圖片預覽
- 標示每張圖片的編號和位置

### 第四步：下載
- 單張下載：點選任意一張單獨下載
- 批次下載：一鍵下載全部圖片
- ZIP 下載：打包成 ZIP 壓縮包下載

### 第五步：提醒畫質修復
- 提醒用戶切割後可能損失畫質
- 建議至即夢/豆包進行高清修復

## 工具指引

- 使用 HTML5 Canvas API 進行圖片切割
- 使用 JSZip 函式庫生成 ZIP 壓縮包
- 使用 FileSaver.js 處理檔案下載
- 前端實現，無需後端服務

## 錯誤處理

| 情境 | 處理方式 |
|------|----------|
| 圖片格式不支援 | 提示支援的格式，請用戶轉檔後重試 |
| 圖片尺寸過小 | 警告切割後畫質可能不佳 |
| 圖片尺寸非正方形 | 自動偵測比例並調整切割方式 |
| ZIP 生成失敗 | 改為逐張下載模式 |
| 瀏覽器不支援 Canvas | 提示用戶更換瀏覽器 |

## 使用範例

- 「幫我切割這張 9 宮格圖片」
- 「把這張拼圖分割成單張圖片，打包成 ZIP」
- 「我要把這張 3x3 的圖片切開來分別下載」

## 護欄

- 僅處理圖片檔案，不接受其他格式
- 切割數量上限為 10x10（100 張）
- 不修改原圖內容，僅做分割處理
- 不上傳圖片到外部伺服器，全程本地處理
- 提醒用戶注意版權問題
