# 9宮格圖提示詞生成器

## 功能說明

根據用戶上傳的參考圖片反推場景，生成包含 9 個不同鏡頭的網格影像 AI 提示詞。支援中英文切換，可直接複製使用。

## 工作流程

### 第一步：接收參考素材
- 用戶上傳一張或多張參考圖片
- 分析圖片中的主體、場景、光線、風格

### 第二步：反推場景描述
- 辨識圖片中的人物/物體特徵
- 提取環境、服裝、光線等關鍵元素
- 統一整合為一致的場景基底

### 第三步：規劃 9 個鏡頭
- 為每個鏡頭設計不同角度和場景
- 保持人物/物體、服裝、光線的一致性
- 鏡頭安排涵蓋遠景、中景、特寫等變化

### 第四步：生成提示詞
- 使用標準模板輸出提示詞
- 提供中文原文及英文翻譯
- 確保可直接複製到 AI 圖片生成工具

## 提示詞模板

```
根據[圖片描述]，產生一張具有聚集力的 3x3 網格影像，
包含在同一環境中的 9 個不同的光源鏡頭，
嚴格保持人物/物體、服裝和光線的一致性，
8K 分辨率，16:9 畫幅。

鏡頭01：[場景描述]
鏡頭02：[場景描述]
...
鏡頭09：[場景描述]
```

## 工具指引

- 使用 HTML/CSS/JavaScript 建立互動頁面（若需要網頁版）
- 使用 Canvas API 預覽 3x3 佈局
- 可整合 AI API 進行圖片分析（可選）

## 錯誤處理

| 情境 | 處理方式 |
|------|----------|
| 用戶未上傳圖片 | 請用戶提供參考圖片或文字描述場景 |
| 圖片模糊無法辨識 | 告知用戶，請提供更清晰的參考圖片 |
| 場景描述不夠具體 | 主動追問風格、光線、氛圍等細節 |
| 9 個鏡頭差異過大 | 重新調整，確保一致性 |

## 使用範例

- 「幫我根據這張照片生成 9 宮格提示詞」
- 「我要做一組產品拍攝的 9 宮格，幫我寫 prompt」
- 「反推這張圖片的場景，然後生成 9 鏡頭提示詞」

## 護欄

- 提示詞必須保持 9 個鏡頭的視覺一致性
- 不生成涉及暴力、色情或不當內容的提示詞
- 輸出必須同時提供中文和英文版本
- 不擅自修改用戶指定的風格或主體
- 每個鏡頭描述控制在 1-2 句話以內
