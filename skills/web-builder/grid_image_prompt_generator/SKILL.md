# 9宮格圖提示詞生成器

## 功能需求

| 功能 | 說明 |
|------|------|
| 圖片上傳 | 支援一張或多張參考圖片上傳 |
| 反推場景 | 根據上傳的圖片反推場景提示詞 |
| 場景選擇 | 可選擇不同分鏡的場景類別 |
| 中英文切換 | 生成的提示詞可切換語言 |
| 9宮格預覽 | 顯示3x3網格佈局預覽 |

## 提示詞模板

```
根據[圖片描述]，產生一張具有聚集力的[3x3]網格影像，
包含在同一環境中的[9]個不同的光源鏡頭，
嚴格保持人物/物體、服裝和光線的一致性，
[8K]分辨率，[16:9]畫幅。

鏡頭01：[場景描述]
鏡頭02：[場景描述]
鏡頭03：[場景描述]
鏡頭04：[場景描述]
鏡頭05：[場景描述]
鏡頭06：[場景描述]
鏡頭07：[場景描述]
鏡頭08：[場景描述]
鏡頭09：[場景描述]
```

## 輸出格式

- 原文（中文）
- 英文翻譯
- 可複製的提示詞

## 技術實現

- HTML/CSS/JavaScript
- Canvas API 預覽
- AI API 整合（可選）
