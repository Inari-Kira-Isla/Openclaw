# Learning Feedback Article Skill

## 觸發條件
當用戶說：
- "生成學習反饋"
- "建立優化網絡"
- "建造文案系統"
- "學習反饋系統"
- "優化網絡文案"

## 功能
根據系統運行數據生成學習反饋文案，包括：
1. 數據收集與分析
2. 模式識別與建議
3. 文案生成與存儲
4. 反饋學習閉環

## 生成流程
1. 收集數據 → 系統運行、錯誤記錄、成功案例
2. 分析模式 → 錯誤模式、成功因素
3. 生成文案 → 技術指標、能力提升、未來展望
4. 反饋學習 → 建立閉環、持續優化

## 輸出格式
- 技術指標表格
- 能力提升分析
- 未來展望規劃
- 經驗總結
