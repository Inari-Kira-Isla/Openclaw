# Data Growth Monitor

## 監控指標

### 容量指標
- 總容量
- 已使用
- 可用空間
- 使用百分比

### 成長指標
- 每日成長量
- 每月成長率
- 成長趨勢

## 閾值設定

| 層級 | 使用率 | 動作 |
|------|--------|------|
| 正常 | < 60% | 無 |
| 警告 | 60-80% | 通知 |
| 危險 | 80-90% | 警報 |
| 緊急 | > 90% | 緊急處理 |

## 報告格式

```json
{
  "current_size": "50GB",
  "growth_rate": "2GB/天",
  "predicted_full": "25天後",
  "recommendations": ["清理舊資料", "擴容"]
}
```
