# Conversation Logger

## 概述

記錄所有 Facebook Messenger 對話，存儲到 Notion 資料庫並提供檢索功能。

## 記錄內容

### 基本資訊

| 欄位 | 描述 |
|------|------|
| 用戶姓名 | FB 顯示名稱 |
| 用戶 ID | FB 用戶 ID |
| 粉絲專頁 | 接收訊息的專頁 |
| 對話時間 | 開始對話時間 |
| 結束時間 | 對話結束時間 |
| 狀態 | 進行中/已關閉/轉接 |

### 對話內容

- 用戶訊息（原文）
- AI 回覆（原文）
- 對話輪次
- 對話摘要

### 分析資料

- 問題類型
- 情緒分析
- 處理結果
- 滿意度

## 資料庫結構

```
對話記錄資料庫
├── 用戶資訊
│   ├── 用戶姓名
│   ├── 用戶 ID
│   └── Messenger URL
├── 對話內容
│   ├── 訊息列表
│   ├── 對話摘要
│   └── 對話標籤
├── 分析
│   ├── 問題類型
│   ├── 情緒分數
│   └── 處理結果
└── 追蹤
    ├── 狀態
    ├── 負責人
    └── 跟進記錄
```

## 同步流程

```
1. 新訊息到達
2. 解析訊息內容
3. 建立/更新用戶檔案
4. 添加訊息到對話串
5. 觸發分析（如需要）
6. 更新對話狀態
```

## 檢索功能

### 按用戶檢索
- 輸入用戶 ID 或姓名
- 查看完整對話歷史

### 按時間檢索
- 設定日期範圍
- 匯出區間對話

### 按標籤檢索
- 問題類型
- 處理狀態
- 滿意度

## 導出格式

支援匯出：
- Markdown
- JSON
- CSV
- PDF

## 隱私保護

- 自動遮蔽敏感資訊
- 遵循 GDPR 規範
- 用戶可要求刪除記錄
