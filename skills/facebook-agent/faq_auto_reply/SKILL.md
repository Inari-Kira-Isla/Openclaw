# FAQ Auto Reply

## 概述

自動回覆粉絲專頁 Messenger 常見問題，減少人工回覆負擔。

## 問題匹配邏輯

### 關鍵詞匹配

| 關鍵詞 | 回覆模板 |
|--------|----------|
| 價格/多少錢/報價 | 價格資訊模板 |
| 怎麼買/如何購買/購買 | 購買流程模板 |
| 運費/配送/寄送 | 配送資訊模板 |
| 退換/退款/退货 | 退換貨模板 |
| 尺寸/大小/規格 | 產品規格模板 |
| 庫存/有沒有貨 | 庫存查詢模板 |

### 模糊匹配

使用 AI 理解用戶意圖，匹配最佳答案。

## 回覆模板

### 價格詢問
```
💰 感謝您的詢問！

我們的產品價格如下：
- 產品 A: $XXX
- 產品 B: $XXX
- 組合優惠: $XXX

如需詳細資訊，請告訴我您感興趣的產品！
```

### 購買流程
```
🛒 購買流程很簡單！

1. 選擇您喜歡的產品
2. 告訴我們數量與規格
3. 我們會提供付款資訊
4. 確認後 24 小時內出貨

有任何問題歡迎隨時詢問！
```

### 物流配送
```
📦 配送資訊：

- 台灣本島：運費 $80，滿 $999 免運
- 港澳：運費 $150
- 海外：依實際材積計算

出貨後約 2-3 天送達！
```

## 轉接條件

以下情況自動轉接人工：

1. 問題不在 FAQ 範圍內
2. 客戶要求與真人對話
3. 多次詢問相同問題
4. 抱怨或投訴類訊息
5. 訂單緊急問題

## 學習機制

每次互動後記錄：
- 用戶問題
- 匹配答案
- 用戶反饋（滿意/不滿意）

用於優化 FAQ 資料庫。
