# GSCC Classifier

## GSCC 分級定義

| 等級 | 風險 | 描述 | 範例 |
|------|------|------|------|
| T0 | 最低 | 純閱讀、查詢、整理 | 查資料、讀文件、搜尋 |
| T1 | 低 | 可回復、可撤銷的操作 | 發訊息、編輯檔案、創建任務 |
| T2 | 中 | 涉及金錢、帳號、隱私 | 付款、帳號設定、敏感資料 |
| T3 | 高 | 不可逆或後果重大 | 刪除資料、系統修改、授權 |

## 評估維度

### 1. 可逆性
- 完全可逆 → T0/T1
- 部分可逆 → T1/T2
- 不可逆 → T2/T3

### 2. 影響範圍
- 僅自己 → T0/T1
- 團隊/組織 → T2
- 系統全局 → T3

### 3. 敏感度
- 公開資訊 → T0
- 內部資訊 → T1
- 隱私/機密 → T2/T3

### 4. 金錢關聯
- 無關 → T0/T1
- 小額 → T2
- 大額/財務 → T3

## 決策矩陣

```
           │ 可逆   │ 部分可逆 │ 不可逆
───────────┼────────┼─────────┼────────
僅自己     │  T0    │   T1    │   T2
團隊       │  T1    │   T2    │   T3
全局       │  T2    │   T3    │   T3
```

## 輸出格式

```json
{
  "grade": "T0|T1|T2|T3",
  "reason": "評估理由",
  "factors": ["factor1", "factor2"],
  "requires_approval": true/false,
  "fallback": "降級時的建議"
}
```

## 特殊規則

- 涉及「刪除」→ 至少 T2
- 涉及「金錢」→ 至少 T2
- 涉及「系統修改」→ 至少 T2
- 涉及「授權/權限」→ 至少 T2
