# 結果整合

將多個 Agent 的輸出去重、合併、格式化，產出結構清晰的統一回覆。

## 操作 / 工作流程

1. **接收結果** — 收集各 Agent 輸出，標記來源
2. **分類標記** — 依內容類型分類（任務完成 / 報告 / 決策）
3. **去重合併** — 相同內容保留一份，相似內容合併，保留差異觀點
4. **術語統一** — 統一用詞、格式、風格
5. **輸出優化** — 依用途選擇模板，重要內容優先排列

## 參數

| 參數 | 類型 | 預設 | 說明 |
|------|------|------|------|
| results | array | — | 各 Agent 的輸出列表 |
| output_type | string | auto | 輸出模板：task / report / decision / auto |
| language | string | zh-TW | 輸出語言 |

## 輸出格式

**任務完成型：**
```
✅ 任務完成
[結果摘要]
📋 詳情：
1. [項目一]
2. [項目二]
```

**報告型：**
```
📊 [報告標題]
[執行摘要]
📈 詳細內容：
[結構化內容]
```

**決策型：**
```
🎯 建議：[最終建議]
📊 分析：[優劣勢]
✅ 下一步：[具體行動]
```

## 錯誤處理

| 錯誤 | 處理 |
|------|------|
| 結果列表為空 | 回報無可整合內容 |
| 格式無法辨識 | 以純文字模式整合，標記警告 |
| 內容互相矛盾 | 保留雙方觀點，標記衝突交由 final_verdict 處理 |

## 使用範例

- "把這些結果整合成一份報告"
- "幫我合併這幾個 agent 的輸出"
- "統一格式後回覆給我"
