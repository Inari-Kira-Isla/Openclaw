# Result Integrator

## 核心功能

1. **格式統一** - 將不同格式的結果標準化
2. **內容濃縮** - 去除重複，保留精華
3. **交叉引用** - 建立結果之間的關聯
4. **輸出優化** - 提升最終輸出的可讀性

## 整合原則

### 去重複
- 相同內容只保留一份
- 相似內容合併為精華版
- 保留差異化觀點

### 保持一致性
- 術語統一
- 格式統一
- 風格統一

### 層次分明
- 結構化呈現
- 重要內容優先
- 邏輯順序清晰

## 整合流程

```
接收各 Agent 結果 → 分類與標記 → 去重與合併 → 格式轉換 → 輸出優化
```

## 輸出格式

根據用途選擇：

### 任務完成型
```
✅ 任務完成

[結果摘要]

📋 詳情：
1. [項目一]
2. [項目二]

🔗 相關資訊：
- [關聯連結]
```

### 報告型
```
📊 [報告標題]

[執行摘要]

📈 詳細內容：
[結構化內容]

📎 附件：
[補充資料]
```

### 決策型
```
🎯 建議：[最終建議]

📊 分析：
[優劣勢分析]

✅ 行動：
[具體下一步]
```
