# Information Gathering

## 收集範圍

### 1. 主題研究
- 產業趨勢
- 競爭分析
- 技術資訊

### 2. 資料搜尋
- 網路搜尋
- 文件查找
- 數據收集

### 3. 資訊驗證
- 來源可靠性
- 數據準確性
- 最新更新

## 輸出格式
```
📚 [主題] 研究摘要

來源：
- [來源1]
- [來源2]

重點：
1. [重點1]
2. [重點2]

建議閱讀：
- [連結1]
```
