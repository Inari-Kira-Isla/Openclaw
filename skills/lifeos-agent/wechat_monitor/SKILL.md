# WeChat Monitor

## 概述

監控 WeChat 網頁版，定時截圖並偵測新訊息，透過 Telegram 通知用戶。

## 功能

### 1. 定時截圖
- 每隔一段時間截取 WeChat 網頁版畫面
- 記錄對話列表狀態
- 追蹤未讀訊息數量

### 2. 新訊息偵測
- 偵測對話列表的新訊息
- 識別發訊人與內容關鍵詞
- 判斷是否需要立即通知

### 3. Telegram 通知
- 有新訊息時立即通知
- 包含訊息摘要
- 點擊可查看詳細內容

## 使用方式

### 啟動監控
```
打開 WeChat 網頁版：https://web.wechat.com
保持登入狀態
告訴我「開始監控 WeChat」
```

### 監控參數
| 參數 | 預設值 | 說明 |
|------|--------|------|
| 截圖間隔 | 60 秒 | 每分鐘檢查一次 |
| 新訊息通知 | 開啟 | 有新訊息時通知 |
| 安靜時段 | 22:00-07:00 | 夜間不通知 |

## 通知格式

### 新訊息通知
```
🔔 WeChat 新訊息

來自：XXX
內容：訊息摘要...
時間：7:48 AM
```

## 風險提醒

- WeChat 網頁版可能突然登出
- 需要保持網頁開啟狀態
- 僅供監控，不自動回覆

## 依賴

- WeChat 網頁版保持開啟
- Telegram 通知（已有）
