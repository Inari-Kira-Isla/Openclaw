# Daily Scan

## 掃描內容

### 1. 郵件
- 未讀郵件
- 重要郵件
- 會議邀請

### 2. 日程
- 今日行程
- 明日預約
- 會議時間

### 3. 待辦
- 今日待辦
- 逾期任務
- 截止提醒

### 4. 資訊
- 天氣
- 交通
- 新聞摘要

## 輸出格式
```
🌅 每日簡報 - [日期]

📧 郵件：X 未讀
📅 行程：X 個
✅ 待辦：X 項

☀️ 天氣：晴，22°C
🚇 交通：順暢
```
