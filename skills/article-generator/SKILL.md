# Article Generator Skill

## 觸發條件
當用戶說：
- "生成文章"
- "寫文章"
- "文章內容"
- "給我文章"
- "需要文章"
- "幫我寫"

## 功能
根據需求生成完整文章，包括：
1. 理解需求並規劃大綱
2. 收集相關資料
3. 撰寫內容
4. 格式化輸出

## 生成流程
1. 理解需求 → 確定主題與目標讀者
2. 資料收集 → 檢索相關記憶與知識
3. 大綱規劃 → 設計章節結構
4. 內容撰寫 → 填充詳細內容
5. 格式排版 → 優化輸出格式

## 輸出格式
- Markdown 格式
- 大綱先行
- 結構清晰
- 重點突出

## 存儲位置
生成的文章自動存儲到：
~/.openclaw/workspace/memory/
