### 評分系統 v2

```
總分 = 好奇心(25%) + 共鳴度(25%) + 價值感(20%) + 行動力(15%) + 情感度(15%)
```

**評分維度：**
- 好奇心：能否引起興趣
- 共鳴度：是否觸動情感
- 價值感：是否有用
- 行動力：是否促使點擊
- 情感度：情緒強度

### 鈎子公式庫

**社群媒體鈎子：**
- "大多數人不知道..."
- "我在___失敗後才明白..."

**銷售文案鈎子：**
- "給___的最後機會"

- "你知道___嗎？"
- "如果你正在___..."

**影片鈎子：**
- "這個影片會改變你對___的看法"
- "99% 的人都錯過了..."
- "看完這個你就知道..."

## 模組二：網絡文案生成器 v2

### 8 種結構模板

1. **痛點→方案→證據**
   - 描述問題 → 提供解決 → 展示結果
   - 適用：教學、產品推廣

2. **現象→分析→建議**
   - 觀察現象 → 深入分析 → 具體建議
   - 適用：趨勢分析、評論

3. **經歷→領悟→分享**
   - 個人故事 → 學到什麼 → 建議讀者
   - 適用：個人品牌、經驗分享

4. **問題→解答→行動**
   - 讀者困惑 → 專業解答 → 呼籲行動
   - 適用：FAQ、專業知識

5. **對比→選擇→結論**
   - 選項比較 → 分析利弊 → 建議選擇
   - 適用：比較文、評測

6. **趨勢→洞察→預測**
   - 現象觀察 → 深度洞察 → 未來預測
   - 適用：趨勢分析

7. **揭秘→證據→方法**
   - 揭示真相 → 證據支持 → 提供方法
   - 適用：乾貨知識

8. **故事→教訓→行動**
   - 案例故事 → 核心教訓 → 行動呼籲
   - 適用：說服力文案

### 去 AI 味技巧 v2

**詞彙替換：**
| AI 常用 | 自然表達 |
|---------|----------|
| 首先、其次、最後 | 然後、接著、最後 |
| 此外、另外 | 還有、而且 |
| 總結來說 | 說白了、講白了 |
| 眾所周知 | 大家都曉得 |
| 值得注意的是 | 說實話 |

**句式優化：**
- 短句優先
- 主動語態
- 具體細節
- 個人觀點

**風格調整：**
- 加入真實案例
- 使用對話感
- 適當留白
- 節奏變化

## 模組三：學習反饋系統 v2

### 追蹤指標 v2

| 指標 | 說明 | 目標 | 計算方式 |
|------|------|------|----------|
| 觸及人數 | 曝光量 | >1000 | 貼文曝光次數 |
| 互動率 | 參與度 | >5% | (讚+留言+分享)/觸及 |
| 點擊率 | 轉化 | >3% | 點擊次數/觸及 |
| 留言率 | 深度互動 | >1% | 留言數/觸及 |
| 分享率 | 傳播力 | >2% | 分享數/觸及 |
| 儲存率 | 收藏度 | >1% | 儲存數/觸及 |
| 觀看完成率 | 影片 | >50% | 看完人數/觀看人數 |
| 回覆率 | 客服質量 | >80% | 回覆數/留言數 |

### 追蹤頻率 v2

| 頻率 | 內容 | 行動 |
|------|------|------|
| 即時 | 異常波動 | 警報 |
| 每小時 | 熱度趨勢 | 優化調整 |
| 每日 | 基本數據 | 記錄分析 |
| 每週 | 深入分析 | 策略調整 |
| 每月 | 全面回顧 | 規劃制定 |
| 每季 | 趨勢分析 | 目標設定 |
| 每年 | 年度總結 | 方向調整 |

**競爭分析：**
