## 腳本鉤子

### 自動化腳本位置
- `~/.openclaw/scripts/closed_loop_system.py` - 閉環系統主腳本
- `~/.openclaw/scripts/agent_closed_loop.py` - Agent 閉環
- `~/.openclaw/scripts/research_hooks.py` - 研究鉤子

### 執行命令
```bash
# 手動執行閉環
python3 ~/.openclaw/scripts/closed_loop_system.py

# 研究鉤子
python3 ~/.openclaw/scripts/research_hooks.py
```

## 觸發條件

### 定時觸發 (Cron)
- 每小時：閉環週期
- 每15分：任務處理
- 每天 09:00：每日計劃

### 事件觸發
- 錯誤檢測 → 觸發錯誤分析
- 任務完成 → 觸發結果驗證
- 新知識 → 觸發知識討論

### 閾值觾發
- 錯誤數 > 3 → 觸發錯誤預警
- 成功率 < 80% → 觸發優化
