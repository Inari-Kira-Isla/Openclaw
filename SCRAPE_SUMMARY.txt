API ENDPOINTS & METHODOLOGY
================================================================================

Base URL: https://www.macaotourism.gov.mo

Restaurants:
  LIST:   GET /api/dining/qtsas_restaurant?lang=zh-hant
  DETAIL: GET /api/dining/qtsas_restaurant/{ID}?lang=zh-hant

Travel Agencies:
  LIST:   GET /api/travelagency?lang=zh-hant
  DETAIL: GET /api/travelagency/{ID}?lang=zh-hant

Scraping Method:
  1. Fetch list endpoints to get all IDs and counts
  2. Iterate through each record to fetch full details
  3. Extract contact information from detail endpoints
  4. Parse awards, hours, and metadata
  5. Export to CSV for MODI integration

Rate Limiting: None observed (0.1s delay between requests used)
