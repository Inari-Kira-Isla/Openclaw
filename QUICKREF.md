# Kira 快速指令參考

## 召喚 Agents

| 指令 | 功能 |
|------|------|
| `叫 knowledge-agent 搜尋 XXX` | 知識庫搜尋 |
| `叫 memory-agent 整理對話` | 記憶整理 |
| `叫 code-master 審查 XXX` | 程式碼審查 |
| `叫 lifeos-agent 查天氣` | 生活助理 |
| `叫 analytics-agent 分析 XXX` | 數據分析 |

## 系統操作

| 指令 | 功能 |
|------|------|
| `/status` | 查看狀態 |
| `/subagents list` | 查看子 Agents |
| `/subagents kill all` | 停止所有子 Agent |

## 搜尋

| 指令 | 功能 |
|------|------|
| `搜尋 XXX` | Web 搜尋（需要 API Key）|
| `查記憶 XXX` | 搜尋記憶庫 |

## 工具

| 工具 | 功能 |
|------|------|
| `web_fetch` | 抓取網頁 |
| `memory_search` | 搜尋記憶 |
| `sessions_spawn` | 召喚子 Agent |
