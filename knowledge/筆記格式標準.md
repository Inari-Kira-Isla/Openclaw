# 筆記格式標準

## 完整筆記結構

每次記錄到 Notion 時，必須包含以下欄位：

### 1. 標題 (Title)
- 簡潔明確的標題
- 可加上 emoji 表情符號

### 2. 類型 (Type) — 單選
- 學習筆記
- 系統架構
- 專案
- 設計
- 工具
- 數據記錄
- 工作流程
- Agent設計

### 3. 標籤 (Tags) — 多選
- Agent
- 架構
- 工作流
- 技能
- 數據
- 分析
- 專案
- 遊戲
- 進度
- 技能系統
- SD
- 角色設計
- AI

### 4. 摘要 (Summary)
- 3-5 行的簡短概述
- 重點：誰、什麼、何時、為什麼

### 5. 詳細內容
- 使用標題層次 (H1, H2, H3)
- 項目符號列表
- 必要時加入 Key-Value 對

### 6. 狀態 (Status) — 單選
- 草稿
- 進行中
- 已完成

---

## 範例

**標題:** 🤖 Claude Sonnet 4.6 發布

**類型:** 學習筆記

**標籤:** AI, Claude, 模型更新

**摘要:** 
Anthropic 發布 Claude Sonnet 4.6，升級重點：
- 支援 1M Token Context
- 程式碼編寫能力提升 70%
- Claude in Excel 全面支援 MCP

## 重點摘要
### Claude Sonnet 4.6
- Context Window: 1M Token
- 程式碼: 70% 測試者偏好
