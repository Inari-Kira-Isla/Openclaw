## Skills 架構應用

### 1. 素材蒐集引擎 (對應 OpenClaw)

```yaml
Skill: web_content_collector
Trigger: "收集" + 主題關鍵字
Actions:
  - 搜尋相關資訊
  - 驗證來源
  - 提取重點
  - 存儲到記憶
```

### 2. 題目篩選系統

```yaml
Skill: topic_filter
Trigger: "篩選題目" + 主題
Actions:
  - 評估熱度
  - 檢查獨家性
  - 預測傳播潛力
  - 給出分數
```

### 3. 爆款內容生成器

```yaml
Skill: viral_content_generator
Trigger: "生成" + 主題
Actions:
  - 結構化大綱
  - 生成多版本
  - 優化標題
  - 添加 CTA
```
