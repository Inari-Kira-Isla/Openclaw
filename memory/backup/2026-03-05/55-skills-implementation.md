## 🎯 核心原則

### 1. 原子化
> 把工作拆解成最小執行單位

### 2. 具體痛點
> 解決操作超過 5 次的問題

### 3. 標準化
> 流程固化、代碼化

### Step 3: 技能模板

```yaml
# SKILL.md 模板
name: skill_name
description: 解決什麼痛點
frequency: 每週/每天/按需

# 輸入
input: 什麼觸發

# 輸出
output: 產出什麼

# 流程
steps:
  - step 1
  - step 2

# 範例
examples:
  - "執行 xxx"
```

---

### Step 4: 建立流程

```
痛點識別 → 原子化 → 寫成 Skill → 測試 → 自動化
```
