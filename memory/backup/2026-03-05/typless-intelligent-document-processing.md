# Typless - 智能文檔處理

**日期**: 2026-02-20
**官網**: https://typless.com

---

## 功能

**Intelligent Document Processing (IDP)**
- 從 PDF、圖片中提取數據

---

## 支援格式

| 格式 | 說明 |
|------|------|
| PDF | 文件 |
| JPG | 圖片 |
| BMP | 圖片 |
| PNG | 圖片 |
| TIFF | 圖片 |

---

## 限制

- 最大檔案: 30MB
- 建議解析度: 300 DPI
- 最大解析度: 600 DPI (A4)
- 不支援手寫發票
- 不支援加密檔案

---

## 應用場景

1. **發票處理** - 自動提取發票數據
2. **表格識別** - 結構化數據提取
3. **文檔歸檔** - 數字化文件
