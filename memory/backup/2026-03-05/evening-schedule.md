# 🌙 晚間排程系統 (Evening Schedule)

## 學習重點

從花生醬團隊學到的：
- 凌晨精進討論 (AI 自動檢討會)
- 每天回顧 + 改進
- 沒有情緒、持續優化

---

## 📋 晚間流程

### 18:00 - 一天結束回顧

| 項目 | 內容 |
|------|------|
| 完成事項 | 今天做到了什麼？ |
| 問題記錄 | 出錯了什麼？ |
| 學習點 | 學到了什麼？ |
| 改進點 | 明天要優化什麼？ |

### 晚間 AI 討論會

**參與者：**
- 花生醬 (總管)
- 海豚 (行銷)
- 草莓醬 (客服)
- 史萊姆 (優化)

**討論問題：**
1. 今天哪裡做得最好？
2. 今天哪裡出了問題？
3. 問題的根本原因是什麼？
4. 明天要改進什麼？
5. 有沒有新知識要記住？
