## 核心觀點

### 1. AI 顯學的轉變

| 時代 | 重點 | 狀態 |
|------|------|------|
| 2024-2025 | Prompt 提示詞 | 過時 |
| 2026 | Skill 技能系統 | 現在進行式 |
| 未來 | Agent | 即將到來 |

### 2. Skill vs Prompt

| 維度 | Prompt | Skill |
|------|--------|-------|
| 可迭代 | ❌ | ✅ |
| 可交易 | ❌ | ✅ |
| 可封裝 | ❌ | ✅ |
| 系統化 | ❌ | ✅ |

### 3. Agent 的本質

```
人類 → 劃分任務 → Agent A (Skill X)
              → Agent B (Skill Y)  
              → Agent C (Skill Z)
              
人類角色 = Harness Engineer (驯服者)
```

---

## 律師事務所的案例

### 傳統模式
- 數百名律師的大型事務所
- 初級律師處理基礎文件工作
- 高成本結構

### AI 模式
- 精品事務所（2人）對抗大型事務所
- AI 處理基礎文件、初稿、研究
- 人類專注判斷力與客戶關係
- 成本減半，品質相同

---

## 關鍵啟示

### 判斷力是人類的最後堡壘

| AI 擅長 | 人類擅長 |
|---------|----------|
| 文件審查 | 判斷何時讓步 |
| 初步研究 | 讀懂空氣、觀察臉色 |
| 草擬初稿 | 處理複雜談判 |
| 格式標準化 | 最終決策負責 |

### 從 MaaS 到 SaaS

- **MaaS**：模型即服務（Model as a Service）
- **SaaS**：技能即服務（Skill as a Service）
