## GitHub + Vercel 自動部署

### 流程

1. **本地生成** → `content/` 或 `public/`
2. **Git Push** → 推送到 `Inari-Kira-Isla/Openclaw`
3. **Vercel 觸發** → 自動偵測變更並部署
4. **網站上線** → https://openclaw.vercel.app

### Vercel 設定

| 項目 | 值 |
|------|-----|
| Framework | Other |
| Build Command | (留空) |
| Output Directory | public |

## 每日自動化腳本

### aeo_daily.py

```python
#!/usr/bin/env python3
"""每日 AEO 自動化"""

import subprocess
import sys

def main():
    steps = [
        ("研究趨勢", "python3 scripts/content_gap_analysis.py"),
        ("生成文章", "python3 scripts/aeo_content.py generate --type random"),
        ("SEO 審計", "python3 scripts/seo_audit.py"),
        ("部署發布", "bash scripts/deploy.sh"),
    ]
    
    for name, cmd in steps:
        print(f"🔄 {name}...")
        result = subprocess.run(cmd, shell=True)
        if result.returncode != 0:
            print(f"❌ {name} 失敗")
            sys.exit(1)
        print(f"✅ {name} 完成")
    
    print("🎉 每日自動化完成!")

if __name__ == "__main__":
    main()
```
