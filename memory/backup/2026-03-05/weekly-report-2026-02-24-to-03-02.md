## 五、學習與洞察

### 5.1 本週學習要點

1. **Agent 設計模式**: Skills over Agents
   - 用 SKILL 封裝領域知識，而非直接 build Agent
   - SKILL 是數位資產，可驅動多個 Agent

2. **閉環系統價值**
   - 從「做事」到「持續優化做事的方式」
   - 自動化 > 手動優化

3. **話題選擇策略**
   - 相關性權重 > 熱門程度
   - 與 OpenClaw 生態系相關的議題更有效

4. **記憶體管理**
   - Context 維持在 20-25% 最優
   - Cache 命中率 60%+ 為目標
