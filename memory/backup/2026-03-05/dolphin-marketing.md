---

## 🎯 核心公式

**衝突型 Hook**
```
[對手]說[質疑]，直到我[展示]，[對方改觀]

範例：
• 「仲介說這個價格不可能再低，直到我拿出這份數據，他沉默了。」
• 「所有人說 45 歲太晚買房，直到他算清楚這張表，改變了主意。」
```

## 🔧 衝突類型資料庫

| 類型 | 範例 |
|------|------|
| 價格衝突 | 「太貴了」vs 「其實可以省」 |
| 年齡衝突 | 「太老了」vs 「現在開始」 |
| 專業衝突 | 「專家說」vs 「我證明了」 |
| 期待衝突 | 「以為」vs 「實際」 |
| 身份衝突 | 「普通人」vs 「做到的」 |

## 📝 Hook 模板

### 房產類
```
• 房仲說不可能...直到我給他看這個
• 所有專家說...但我實際做了發現...
• 以為買房要...結果...
```

### 創業類
```
• 別人說創業要...我卻...
• 所有人都說...直到...
• 以為...原來...
```

### 投資類
```
• 銀行說...但數據告訴我...
• 專家說...我親自驗證後發現...
• 大家都說...只有我知道...
```
