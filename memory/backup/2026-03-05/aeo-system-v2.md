# AEO 批量生成系統 v2.0

**目標**: 每日 10+ 篇高質量文章  
**建立日期**: 2026-03-02

---

## 架構

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│ content-collector │ ──▶ │ article-generator │ ──▶ │ seo-auditor    │
│ (每小時熱門話題)   │     │ (批量生成)        │     │ (發布前審計)    │
└─────────────────┘     └──────────────────┘     └─────────────────┘
                                                        │
                                                        ▼
                                                ┌─────────────────┐
                                                │ github-publisher │
                                                │ (自動上傳)       │
                                                └─────────────────┘
```

---

## 模組

### 1. content-collector
| 項目 | 內容 |
|------|------|
| 功能 | 每小時趨勢收集 + 關鍵字分析 |
| 輸入 | HN, Reddit, Perplexity, Google Trends |
| 輸出 | 5-10 個熱門話題 + 關鍵字建議 |
| 排程 | 每小時 |

### 2. article-generator (批量)
| 項目 | 內容 |
|------|------|
| 功能 | 根據熱門話題批量生成 AEO 文章 |
| 輸入 | content-collector 輸出 |
| 產出 | 10+ 篇/日 |
| 品質 | SEO ≥80, 字數 1500-3000 |

### 3. seo-auditor
| 項目 | 內容 |
|------|------|
| 功能 | 發布前自動審計 |
| 檢查 | H1, keywords, description, structure |
| 通過率 | 目標 90%+ |

### 4. github-publisher
| 項目 | 內容 |
|------|------|
| 功能 | 自動上傳 GitHub Pages |
| 排程 | 每日 2 次 (10:00, 16:00) |
| 成功率 | 目標 100% |

### 5. learning-loop
| 項目 | 內容 |
|------|------|
| 功能 | 成效追蹤 → 優化提示詞 |
| 週期 | 每週閉環 |
| 指標 | 搜尋排名, 流量, 轉化 |

---

## 每日目標

| 時間 | 任務 | 產出 |
|------|------|------|
| 09:00 | 趨勢研究 | 10 個話題 |
| 10:00 | 批量生成 #1 | 5 篇文章 |
| 14:00 | 批量生成 #2 | 5 篇文章 |
| 15:00 | SEO 審計 | 通過清單 |
| 16:00 | GitHub 上傳 | 發布完成 |
| 每週 | 學習閉環 | 優化建議 |

---

## 技術棧

- **生成**: MiniMax-M2.5 / Ollama
- **儲存**: GitHub Pages
- **排程**: OpenClaw Cron
- **監控**: AEO Dashboard
