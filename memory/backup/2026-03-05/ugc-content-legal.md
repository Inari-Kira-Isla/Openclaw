## 合規流程

### 改進後既流程

```
1. 題目確認
   ↓
2. 選擇素材來源
   ┌─────────────────────────────────────┐
   │ ✅ Unsplash / Pexels (免費)         │
   │ ✅ AI 生成 (DALL-E / 本地)          │
   │ ❌ Google 圖片 (唔建議)             │
   └─────────────────────────────────────┘
   ↓
3. 檢查 License
   - 確認商用許可
   - 記錄來源
   ↓
4. 發佈
```

## 檢查清單

### 發佈前檢查

- [ ] 素材來源係邊？
- [ ] 有冇商用許可？
- [ ] 需要標註出處？
- [ ] 平台允許嗎？
- [ ] 涉及人物嗎？(肖像權)

---

## 各平台要求

### TikTok
- 原創內容優先
- 第三方素材需授權
- 音樂需版權許可

### Instagram Reels
- 類似 TikTok
- 音樂庫相對安全

### YouTube Shorts
- Content ID 系統
- 侵權會emonetize
