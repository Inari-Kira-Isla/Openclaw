## 標準化流程

### 1️⃣ 預先規劃
```
□ 選擇主題
□ 研究關鍵字 (Google/Perplexity)
□ 確定目標受眾
□ 設定文章結構
```

### 2️⃣ 內容生成 (遵循 AEO 標準)
```
□ H1 標題 (含關鍵字)
□ Description (150-160字)
□ 關鍵字優化 (密度 1-2%)
□ 結構化資料 (H2/H3)
□ 列表/表格 (增加豐富度)
```

### 3️⃣ SEO 檢查清單
```
□ 單一 H1
□ 圖片 Alt 文字
□ 內部連結
□ 外部連結 (權威來源)
□ 字數: 1500+ 字
```

### 4️⃣ 發布前檢查
```
□ author 欄位
□ category 欄位
□ date 欄位
□ tags 標籤
```
