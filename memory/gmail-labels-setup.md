# Gmail 標籤設定指南

## 步驟 1: 建立標籤

在 Gmail 中建立以下標籤：

### 方法 A: 手動建立
1. 打開 Gmail
2. 左側「設定」→ 「設定」
3. 「標籤」分頁
4. 「建立新標籤」
5. 輸入名稱:
   - `AI/重要` (重要郵件)
   - `AI/待辦` (待辦郵件)
   - `AI/一般` (一般郵件)

### 方法 B: 使用 Gmail Filter 自動標記

建立 Filter:
1. 設定 → Filter
2. 輸入條件
3. 「建立篩選器」
4. 選擇「套用標籤」

## 常用 Filter 規則

| 條件 | 動作 |
|------|-------|
| from:google.com | 標記為重要 |
| subject:緊急 | 標記為待辦 |
| 促銷郵件 | 自動分類到促銷 |

## 標籤列表

```
Gmail 標籤:
├── INBOX (收件匣)
├── IMPORTANT (重要)
├── TODO (待辦)
└── [自訂]
    ├── AI/重要
    ├── AI/待辦
    └── AI/一般
```

## n8n 中使用

在 n8n Gmail 節點使用標籤 ID：
- `IMPORTANT` - 重要
- `TODO` - 待辦
- `INBOX` - 收件匣
