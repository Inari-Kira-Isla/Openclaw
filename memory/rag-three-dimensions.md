# RAG 三維度配置

## 1. 延伸 (Extension)

### Chunk 策略
| 參數 | 數值 | 說明 |
|------|------|------|
| chunk_size | 500-1000 | 根據內容調整 |
| overlap | 50-100 | 保持上下文 |

### 向量化覆蓋
- 每個概念多角度向量化
- 關鍵詞 + 語義 + 上下文

## 2. 深度 (Depth)

### 語義層次
| 層次 | 目標 |
|------|------|
| 表層 | 關鍵字匹配 |
| 語義 | 語境理解 |
| 意圖 | 推理預測 |

### 雙模型驗證
- Ollama → MiniMax → 確認

## 3. 關聯 (Relationships)

### 相似度閾值
| 參數 | 數值 |
|------|------|
| min_score | 0.5 |
| expand_threshold | 0.3 |

### 跨領域連接
- 概念圖譜
- 因果鏈
- 類比關係
