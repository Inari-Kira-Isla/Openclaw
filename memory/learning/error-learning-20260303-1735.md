### 錯誤上下文


### 原因分析

1. **1006 錯誤**: Node 與 Gateway 之間的 WebSocket 連接被異常中斷
   - 可能原因：網路波動、節點負載過高、Gateway 服務不穩定
   
2. **1012 錯誤**: Gateway 服務主動重啟
   - 預期行為：服務重啟時會短暫中斷連接
   
3. **4000 錯誤**: 心跳超時
   - 可能原因：節點響應不及時
