## 🔧 **技術修改**

### **文章自動生成器 (`site_article_generator.py`)**
**現狀問題**：完全沒有主題過濾，導致生成經濟/政策相關文章

**修改方案**：
```python
# 新增話題白名單
ALLOWED_TOPICS = {
    "AI技術": ["Claude", "ChatGPT", "LLM", "機器學習", "自然語言處理"],
    "工具配置": ["Ollama", "n8n", "Docker", "MCP", "API集成"],
    "應用洞察": ["AI客服", "內容生成", "數據分析", "教育技術"],
}

# 拒絕清單
BLOCKED_TOPICS = [
    "進口關稅", "匯率", "燃油", "天氣", "食品價格",
    "貿易政策", "經濟分析", "外送平台", "法規"
]

def should_generate_article(topic):
    for blocked in BLOCKED_TOPICS:
        if blocked in topic:
            return False
    # ... 進一步驗證
    return True
```
