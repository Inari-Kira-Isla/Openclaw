# Gemini Prompting Guide 101 - 中文寶庫

Google 官方 71 頁《Gemini for Workspace Prompting Guide 101》的完整拆解版本。

147 個實戰 Prompt，按 6 大職能分類，直接複製即用。

## 結構
- `index.html` — 首頁 + 快速導航
- `categories/` — 6 大職能分類頁面
- `prompt-data.json` — 全部 Prompt 結構化數據

## 快速開始
1. 選擇你的職位分類
2. 複製 Prompt 文本
3. 改括號內的資訊，發給 Gemini
4. 得到結果
