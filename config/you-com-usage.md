## 🔍 搜尋條件

### ✅ 使用場景

| 優先級 | 條件 | 範例 |
|--------|------|------|
| **高** | 自我進化訓練 | AI 趨勢、新聞 |
| **高** | 知識庫更新 | 技術資訊 |
| **中** | 事實查證 | 確認最新資訊 |
| **中** | 市場分析 | 競品研究 |
| **低** | 一般查詢 | 日常問題 |

### ❌ 避免場景

| 項目 | 原因 |
|------|------|
| 本地知識庫已有 | 浪費配額 |
| 簡單問題 | MiniMax 可處理 |
| 突發大量需求 | 保留給重要任務 |

## 🎯 觸發條件

### 使用 You.com API 當：

1. **需要最新資訊** (過去 24 小時)
2. **本地知識庫沒有**
3. **需要多元來源**
4. **事實查證**

### 使用本地模型當：

1. **一般分析**
2. **本地知識庫足夠**
3. **創意生成**

### 使用 MiniMax 當：

1. **複雜推理**
2. **高質量輸出**
3. **需要驗證**
