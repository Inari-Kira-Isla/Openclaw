## 🎯 高優先級

### 1. Unsplash API (圖片搜尋)

**官網**: https://unsplash.com/developers

**申請步驟**:
1. 前往 https://unsplash.com/developers
2. 點擊 "Your apps" → "New Application"
3. 填寫資料 (名稱隨便)
4. 接受條款
5. 取得 API Key

**免費額度**: 5,000 次/月

---

### 2. OpenWeatherMap API (天氣)

**官網**: https://openweathermap.org/api

**申請步驟**:
1. 前往 https://openweathermap.org
2. 點擊 "Sign In" → 註冊帳號
3. 進入 "API Keys"
4. 複製 API Key

**免費額度**: 1,000,000 次/月

---

## 🎯 中優先級

### 3. Coqui TTS (文字轉語音)

**官網**: https://coqui.ai/

**安裝方式**:
```bash
# Python
pip install TTS

# 本地運行
tts --text "Hello" --output output.wav
```

**費用**: 開源免費

---

### 4. Cloudinary (圖片優化)

**官網**: https://cloudinary.com/

**申請步驟**:
1. 前往 https://cloudinary.com/users/sign_up
2. 用 Email 註冊
3. 取得 API Key

**免費額度**: 25分鐘 傳輸/個月
