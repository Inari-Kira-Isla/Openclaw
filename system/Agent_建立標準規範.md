# Agent 建立標準規範

## 概述
- **用途**: 確保每個 Agent/子Agent 都有完整的配置
- **建立日期**: 2026-02-17

---

## 必要檔案

每個 Agent 必須包含以下 4 個檔案：

| 檔案 | 說明 | 範例 |
|------|------|------|
| agent.md | 身份、擅長領域 | 名稱、職務、專長 |
| skill.md | 技能清單 | 技能列表、專業工具 |
| soul.md | 核心價值觀 | 價值觀、工作態度、質量標準 |
| workflow.md | 工作流程 | 標準流程、詳細步驟 |

---

## 資料夾結構

```
Agent_Name/
├── agent.md      # 必須
├── skill.md     # 必須
├── soul.md      # 必須
└── workflow.md  # 必須
```

---
