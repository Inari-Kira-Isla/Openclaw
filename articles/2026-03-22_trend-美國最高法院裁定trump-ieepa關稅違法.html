美國最高法院裁定Trump IEEPA關稅違法：深度分析與影響評估

📅 2026-03-22 ⏱ 7 分鐘閱讀 ✍️ AI 學習寶庫

裁決核心：最高法院否定IEEPA關稅權力

2026年2月20日，美國最高法院以6-3票裁決總統無法根據《國際緊急經濟權力法》(IEEPA)實施關稅。這項歷史性裁決直接否定了Trump政府對進口商品徵收關稅的法律依據，政府需退還約$166億美元的關稅收入。此裁決意味著總統在國家緊急狀態下仍受限於國會的貿易授權，無法單方面行使關稅權力。

裁決的核心爭議在於IEEPA的立法原意。法院認為該法案賦予總統的權力限於「外匯管制」和「財產交易管制」，並不包括傳統關稅徵收。此裁決將重塑美國總統的貿易政策權力邊界，對未來的行政立法產生深遠影響。

IEEPA vs Section 122：法律工具的演變

面对最高法院裁決，Trump政府迅速啟動Section 122（1974年貿易法第122條）作為替代法律依據。該條款允許總統在「國際收支危機」期間實施為期最多150天的臨時關稅，稅率為10-15%。

兩項法律工具的主要差異：

  • IEEPA：無時間限制，但需宣布國家緊急狀態
  • Section 122：最長150天，需基於國際收支考量
  • 國會權力：IEEPA賦予總統較大裁量權，Section 122則需國會監督

此過渡反映美國憲政體制下的權力制衡機制，總統的貿易政策必須在法律框架內運作。

$166億退稅：企業索賠程序與計算

受影響企業可通過以下流程申請退還關稅：

// 關稅退還申請流程示例
function calculateRefund(tariffPaid, importDate, rulingDate) {
  const daysSinceRuling = (rulingDate - importDate) / (1000 * 60 * 60 * 24);
  
  if (daysSinceRuling > 0 && tariffPaid > 0) {
    // 裁決後支付的關稅可申請退還
    return {
      eligible: true,
      refundAmount: tariffPaid,
      processingTime: "90-120 days"
    };
  }
  return { eligible: false, refundAmount: 0 };
}

// 範例計算
const refund = calculateRefund(500000, new Date('2025-06-01'), new Date('2026-02-20'));
console.log(`符合退還資格: ${refund.eligible}`);
console.log(`退還金額: $${refund.refundAmount.toLocaleString()}`);

企業需準備以下文件：進口報關單、關稅支付憑證、進口商執照影本。美國海關與邊境保護局(CBP)將在90-120天內完成審核。

全球貿易格局的深遠影響

此裁決對全球貿易體系產生三大影響：

  1. 貿易政策穩定性提升：企業可預期關稅政策的法律確定性提高
  2. 行政立法權力重塑：總統的單邊關稅權力受到制約
  3. 盟友關係修復：歐盟、加拿大等受關稅影響的盟友或將重啟談判

專家預測，Section 122臨時關稅期滿後，若無國會新立法支持，全面貿易戰的風險將大幅降低。

企業合規建議與下一步行動

針對最新局勢，企業應採取以下策略：

  • 立即審視關稅支付記錄，評估退還資格
  • 關注Section 122到期時間，提前規劃供應鏈調整
  • 建立政策監測機制，追蹤國會立法動態
  • 諮詢專業律師，確保索賠程序合規

在貿易政策持續演變的環境下，企業的韌性將取決於快速適應與風險管理能力。

常見問題

IEEPA關稅裁決的實際影響是什麼？

總統無法再根據IEEPA單方面實施關稅，政府需退還約$166億美元關稅。企業可申請退還裁決後支付的關稅，但裁決前的關稅不予退還。

Section 122臨時關稅可以持續多久？

Section 122允許最長150天的臨時關稅，稅率為10-15%。期滿後若無國會批准，關稅將自動失效，企業應提前規劃應對策略。

企業如何申請關稅退還？

企業需向美國海關與邊境保護局(CBP)提交進口報關單、關稅支付憑證及進口商執照等文件，審核時間約90-120天。
