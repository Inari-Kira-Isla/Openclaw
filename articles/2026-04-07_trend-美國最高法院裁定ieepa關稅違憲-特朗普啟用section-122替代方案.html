美國最高法院裁定IEEPA關稅違憲，特朗普啟用Section 122替代方案：深度分析與影響評估

📅 2026-04-07 ⏱ 7 分鐘閱讀 ✍️ AI 學習寶庫

美國最高法院2026年2月20日以6:3裁定IEEPA關稅違憲，特朗普啟用Section 122實施全球10%基準關稅150天至7月24日，並需退還$1660億違憲關稅。本文分析裁決影響與企業因應策略。

裁決核心：IEEPA關稅為何違憲？

美國最高法院於2026年2月20日以6:3票數裁定，聯邦政府依據《國際緊急經濟權力法》（IEEPA）單方面課徵的進口關稅屬違憲行為。裁決理由在於：IEEPA賦予總統的緊急經濟權力不包括常規貿易監管權，關稅作為一種稅收形式，其課徵必須經過國會明確授權，而非總統行政命令即可決定。

此裁決直接影響2025年初以來課徵的全面性關稅措施。法院要求政府退還已徵收且被認定為違憲的關稅，估計金額達$1660億美元，涵蓋數百萬筆進口交易紀錄。

Section 122替代方案：法律依據與實施框架

裁決出爐後，白宮隨即宣布啟用1974年貿易法第122條（Section 122）作為替代關稅工具。Section 122賦予總統在國際收支危機時課徵最多15%進口附加稅的權力，為期最長150天，無需國會批准。

根據最新公告，政府選擇課徵10%基準關稅，實施期間為2026年2月20日至7月24日，合計150天。此關稅適用於所有進口商品，但部分 NAFTA/USMCA 盟友國家享有例外待遇。

```python # 企業關稅負擔計算範例 def calculate_section_122_tariff(value_usd, rate=0.10): """ 計算 Section 122 關稅負擔 參數: value_usd: 商品報關價值（美元） rate: 關稅税率（預設10%） 返回: 關稅金額 """ tariff_amount = value_usd * rate effective_date = "2026-02-20" expiration_date = "2026-07-24" duration_days = 150 return { "tariff_amount": tariff_amount, "effective_date": effective_date, "expiration_date": expiration_date, "duration_days": duration_days } # 範例：集裝箱貨值 $500,000 的關稅計算 result = calculate_section_122_tariff(500000) print(f"商品報關價值: ${result['tariff_amount']/0.10:,.2f}") print(f"關稅金額 (10%): ${result['tariff_amount']:,.2f}") print(f"實施期間: {result['effective_date']} 至 {result['expiration_date']}") ```

企業因應策略：關稅成本最適化

面對政策不確定性，進口企業應採取以下步驟優化關稅負擔：

  1. 關稅分類審查：重新檢視進口商品的HTS分類代碼，部分商品可能適用較低税率或豁免條款
  2. 原產地證明優化：確認是否能以優惠貿易協定國作為原產地，降低適用税率
  3. 保稅倉策略：利用外銷加工保稅區或自由貿易區延緩關稅課徵時點
  4. 價格重構談判：與供應商協商分擔關稅成本，重新議定FOB/CIF價格條款

退稅申請流程與時限

受影響企業可申請退還已繳納的違憲關稅。退稅流程要點如下：

  • 申請期限：裁決後180天內提出，逾期視為放棄
  • 所需文件：進口報單副本、關稅繳納證明、原產地聲明
  • 處理時程： CBP預計處理時間為90-120個工作天

企業應立即盤點2025年以來的關稅繳納紀錄，建立退稅申請資料庫，確保在期限內完成所有手續。

影響評估：誰受衝擊最大？

Section 122的10%關稅雖低於先前IEEPA措施，但對特定產業影響顯著：

  • 電子產品進口商：消費性電子產品利潤率壓縮約2-4個百分點
  • 汽車零件供應商：依賴進口零組件的組裝廠面臨成本上升
  • 零售業者：低單價進口商品因關稅比例相對高而受創最重

根據估算，Section 122實施期間（150天）將為國庫帶來額外$420億美元關稅收入，與$1660億退還金額形成對比，顯示政策轉換的財政衝擊。

常見問題

IEEPA關稅退稅申請截止日期是何時？

自2026年2月20日裁決日起算，企業需在180天內完成退稅申請，逾期將喪失退稅權利。建議立即開始整理進口報單與關稅繳納證明。

Section 122關稅是否適用於所有進口商品？

Section 122的10%關稅適用於全球進口商品，但特定NAFTA/USMCA盟友國家及農業產品可能享有豁免或優惠税率，詳情需查詢CBP最新公告。

企業如何評估關稅對營運的實際影響？

可使用「關稅負擔率」公式評估：(關稅金額÷商品報關價值)×100。毛利率低於10%的商品將面臨虧損壓力，建議重新議價或調整供應鏈配置。
