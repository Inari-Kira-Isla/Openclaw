## 詳細分工

### 一、核心決策 Agent

| Agent | 主要功能 | 次要功能 | 擅長領域 |
|-------|----------|----------|----------|
| **Kira** | 方案提出、推動決策 | 協調各 Agent | 策略規劃、任務分配 |
| **Neicheok** | 第一裁決、審查決定 | 風險評估 | 法律、倫理、決策把關 |
| **Evolution** | 質疑、改進建議 | 質量審查 | 邏輯分析、優化建議 |

**協作流程：**
```
Kira 提出方案 → Neicheok 裁決 → Evolution 質疑 → 最終執行
```
