## 模組詳情

### 1. 記憶層 (@Cynthia)
| 檔案 | `slime-system/memory-layer.md` |
|------|--------------------------------|
| 核心功能 | SQLite-Vec 向量檢索、MEMORY.md 本地筆記、Notion 雲端知識庫 |
| 儲存格式 | 向量嵌入 + 結構化文字 + 標籤分類 |
| 檢索流程 |  Query → Fusion Engine → 多源融合 → 回覆 |

### 2. 學習機制 (@史萊姆)
| 檔案 | `slime-system/learning-mechanism.md` |
|------|--------------------------------------|
| 核心功能 | Prompt Refinement、QA Learning、Template Updater |
| 優化流程 | 收集樣本 → 分析問題 → 生成優化 → 驗證 → 版本記錄 |
| QA 學習 | 新問題識別 → 知識庫檢索 → 答案生成 → 人工審核 → 入庫 |

### 3. 進化引擎 (@Evolution)
| 檔案 | `slime-system/evolution-engine.md` |
|------|-----------------------------------|
| 核心功能 | 漂移偵測、效能分析、自我改進 |
| 偵測類型 | 概念漂移、效能漂移、記憶衰減、行為漂移 |
| 改進策略 | L1 參數調整 → L2 架構優化 → L3 範式替換 |

### 4. 工作流 (@Team)
| 檔案 | `slime-system/workflow.md` |
|------|----------------------------|
| 排程時間 | 02:00記憶整合、08:00/14:00/20:00複習、03:00清理 |
| 狀態機 | PENDING → RUNNING → COMPLETED/FAILED/CANCELLED |
| 核心工作流 | 新規學習輸入、記憶提取與複習、夜間記憶整合 |

### 5. 中央協調 (@Kira)
- 任務分流與調度
- 結果整合與回覆
- 團隊協調
