# 商業案例：Da Bomb - 情緒價值勝利

**日期**: 2026-02-19
**來源**: Joe 分享

---

## 概要

| 項目 | 內容 |
|------|------|
| 創辦人 | 兩個 14 歲少女 (Caroline & Isabel) |
| 產品 | 泡澡球 (Bath Bombs) |
| 年營收 | 6 億台幣 (2000萬美元) |
| 對手 | Lush、Bath & Body Works |

---

## 核心洞見

> **「功能變副作用，情緒才是主產品」**

---

## 商業模式

### 傳統思路
```
成分 → 功能 → 購買
```

### Da Bomb 思路
```
情緒 → 驚喜 → 分享 → 複利
```

---

## 關鍵策略

### 1. 重新定義戰場

| 對手 | Da Bomb |
|------|---------|
| 環保手工 | 開箱驚喜 |
| 香氛體系 | 盲盒體驗 |
| 成分競爭 | 情緒設計 |

### 2. 產品即內容

- 浴球溶解過程 → TikTok 素材
- 「開箱驚喜」→ 用戶自發分享
- 視覺錘效應 → 零成本傳播

### 3. 情緒武器化

```
期待 → 驚喜 → 尖叫 → 分享 → 上癮
```

---

## 商業啟示

### 紅海突圍公式

```
不優化功能 → 重新定義維度 → 讓對手變無關
```

### 核心問題

> 「我的品類裡，有沒有一個被忽視的情緒缺口？」

### 商業真相

- **功能決定成交**
- **情緒決定分享**
- **「多餘的快樂」是最強護城河**

---

## 應用：Lazy Uni Shop

### 情緒價值設計

| 產品 | 功能 | 情緒價值 |
|------|------|----------|
| 寵物用品 | 使用價值 | 陪伴幸福感 |
| 3C 配件 | 便利性 | 科技酷感 |
| 生活家居 | 實用性 | 生活儀式感 |

### 自動化內容策略

```
不只是展示產品 → 展示使用時的情緒體驗
不只是賣東西 → 賣期待感、驚喜感
```

## 名言

> 「當你成功把情緒價值武器化，即便是巨頭壟斷的紅海，也能炸出一片新天地。」

> 「市場從來不缺好產品，缺的是讓人記住、願意分享、會上癮的情緒體驗。」

> 「巨頭之所以輸，不是因為少女比較聰明，是因為巨頭太理性。」

## 行動問題

1. 我的天賦適合填補哪種情緒空位？
2. 我的產品能不能自己長出內容？
3. 能不能把一次性消費變成體驗複利？
