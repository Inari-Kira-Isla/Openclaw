## 📊 數據流

```
輸入：用戶任務
    │
    ▼
muse-core 調度
    │
    ▼
Librarian (Ollama) ──▶ Briefing Report
    │
    ▼
Architect (MiniMax) ──▶ 執行結果
    │
    ▼
memory_deepener ──▶ 深化記憶
    │
    ├──▶ ChromaDB (向量)
    └──▶ Notion (人類可讀)
```

## ⚡ 關鍵特色

| 特色 | 說明 |
|------|------|
| 本地預過濾 | Ollama 先檢索，確保乾淨上下文 |
| 雲端深加工 | MiniMax 深度推理產出結果 |
| 上下文傳遞 | Briefing Report 確保資訊不斷層 |
| 自動深化 | 每次任務後自動提煉 + 儲存 |
| 衝突檢測 | 避免錯誤覆蓋正確知識 |
