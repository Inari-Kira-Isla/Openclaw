## Heartbeat 設定

### 選項 1: 每分鐘 (進階)

```yaml
- name: "模型訓練"
  schedule: "every 1m"
  task: "搜索AI最新資訊並學習"
```

### 選項 2: 每 15 分鐘 (推薦)

```yaml
- name: "自我進化訓練"
  schedule: "every 15m"
  task: "搜索最新AI資訊、評估、優化"
```

### 選項 3: 每小時 (基礎)

```yaml
- name: "知識更新"
  schedule: "every 1h"
  task: "搜索新知識、存入記憶"
```

---

## 每日驗收流程

### 早上 9:00 - 設定目標

```
1. 設定今日學習主題
2. 分配搜尋關鍵詞
3. 啟動自動化
```

### 下午 6:00 - 驗收成果

```
1. 檢查學習記錄
2. 評估模型表現
3. 記錄反饋
4. 調整方向
```

### 晚上 9:00 - 優化

```
1. 分析反饋數據
2. 調整模型參數
3. 更新訓練計劃
```
