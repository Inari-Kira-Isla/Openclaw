## 🧠 優化方式

### 1. 數據驅動優化

| 數據來源 | 優化目標 |
|----------|----------|
| 查詢關鍵詞 | 熱門主題優先 |
| 相關性評分 | 調整權重 |
| 用戶行為 | 學習偏好 |

### 2. 具體優化策略

#### A. 查詢擴展 (Query Expansion)

```
如果用戶經常搜「海膽價格」
→ Ollama 自動理解「海膽價格」≈「海膽市場」「海膽成本」
→ 擴展搜尋範圍
```

#### B. 權重調整 (Weight Tuning)

```
如果「系統架構」評分高
→ 增加「系統」「架構」權重
→ 降低其他詞權重
```

#### C. 結果重排 (Result Reranking)

```
根據歷史評分
→ 高評分文檔排名提升
→ 低評分文檔排名下降
```

## 📊 學習機制

### 短期優化 (每次查詢)

```python
# 立即調整
if user_rating >= 4:
    boost_document_score(doc_id)
elif user_rating <= 2:
    reduce_document_score(doc_id)
```

### 中期優化 (每週分析)

```python
# 每週分析熱門查詢
popular_queries = get_popular_queries()
for query in popular_queries:
    # 預先建立索引
    prebuild_index(query)
```

### 長期優化 (每月複習)

```python
# 每月訓練模型
monthly_data = collect_feedback(month=1)
retrain_model(monthly_data)
```

## 🔧 技術實現

### 1. 反饋收集

```python
# 每次檢索後
feedback.add(
    documents=[query],
    metadatas=[{
        "relevant": user_relevant,
        "rating": user_rating,
        "timestamp": now
    }]
)
```

### 2. 效能分析

```python
# 每週分析
if avg_rating < 3.5:
    # 觸發優化
    expand_query_terms()
    adjust_weights()
    rerank_results()
```

### 3. 自動優化

```python
# 基於反饋自動調整
def optimize():
    # 找出低評分查詢
    low_rated = get_low_rated_queries()
    
    for query in low_rated:
        # 擴展關鍵詞
        expanded = expand_keywords(query)
        # 更新索引
        update_index(expanded)
```
