Claude Desktop 完整配置指南：MCP Server 一鍵安裝教學

📅 2026-03-06 ⏱ 8 分鐘閱讀 ✍️ AI 學習寶庫

Claude Desktop 完整配置指南：MCP Server 一鍵安裝教學

Claude Desktop 是 Anthropic 推出的 AI 桌面應用程式，而 MCP Server（Model Context Protocol Server）是擴展其功能的關鍵工具。透過正確配置 MCP Server，您可以讓 Claude 存取本地檔案、執行系統指令、整合各種開發工具，大幅提升 AI 助理的實用性。本文將帶您一步步完成完整配置。

什麼是 MCP Server？為什麼需要安裝？

MCP Server 是 Claude Desktop 的擴充套件，採用 Model Context Protocol 協定，讓 AI 能與本地系統互動。安裝 MCP Server 後，Claude 可以讀取硬碟檔案、運用指令列工具、查詢資料庫，甚至操控應用程式。這對於需要 AI 協助開發、自動化工作流程的使用者來說是必備功能。

前置條件與環境檢查

開始安裝前，請確認系統已安裝 Node.js（建議版本 18 以上）和 Git。您可以在終端機輸入以下指令檢查：

node --version
npm --version
git --version

如果看到版本號碼表示環境正常，否則請先下載 Node.js進行安裝。

MCP Server 一鍵安裝步驟

以下是最常用的檔案系統 MCP Server 安裝流程：

步驟一：開啟設定檔案

找到 Claude Desktop 的設定檔，路徑如下：

  • macOS：~/Library/Application Support/Claude/claude_desktop_config.json
  • Windows：%APPDATA%\Claude\claude_desktop_config.json
  • Linux：~/.config/Claude/claude_desktop_config.json

步驟二：安裝 MCP Server 套件

開啟終端機，使用 npm 全域安裝所需的 MCP Server：

npm install -g @modelcontextprotocol/server-filesystem

此指令會安裝檔案系統伺服器，讓 Claude 能夠讀取和操作指定目錄的檔案。

步驟三：編輯設定檔

用文字編輯器開啟設定檔，輸入以下內容：

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/Users/你的用戶名/Desktop"]
    }
  }
}

請將/Users/你的用戶名/Desktop替換為您想讓 Claude 存取的目錄路徑。儲存檔案後，關閉並重新開啟 Claude Desktop。

步驟四：驗證安裝

重新啟動 Claude Desktop 後，點擊左下角的齒輪圖示進入設定，確認是否看到 MCP Server 已啟用。您也可以直接詢問 Claude：「請列出我桌面上的檔案」來測試是否正常運作。

進階：安裝多個 MCP Server

您可以同時安裝多個不同的 MCP Server 來擴充功能。以下是常見的伺服器推薦：

  • GitHub Server：整合 GitHub API 操作 Repository
  • Brave Search：即時網路搜尋功能
  • SQLite：本地資料庫查詢

多伺服器設定範例：

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/你的目錄路徑"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"]
    }
  }
}

儲存後重啟 Claude Desktop 即可使用所有已設定的伺服器。

常見問題與解決方案

如果遇到連線失敗的情況，可以嘗試以下解決方式：

  • 確認 npx 指令路徑正確，可使用 which npx 檢查
  • 檢查設定檔的 JSON 語法是否正確，確保沒有漏掉逗號或引號
  • 嘗試將 npx 改為 /usr/local/bin/npx 的完整路徑
  • 查看 Claude Desktop 的 Logs 目錄排查錯誤訊息

結語

完成上述設定後，Claude Desktop 將具備讀取本地檔案、執行系統指令等強大功能。您可以根據工作需求安裝不同的 MCP Server，打造專屬的 AI 開發環境。建議從基礎的檔案系統伺服器開始，逐步探索更多進階功能。

常見問題

MCP Server 安裝後無法運作怎麼辦？

請先檢查 Node.js 版本是否為 18 以上，並確認設定檔 JSON 語法正確。可嘗試在終端機直接執行 npx 指令測試是否正常運作。

如何讓 Claude 存取多個不同目錄？

在設定檔的 args 參數中放入多個路徑即可，例如：["-y", "@modelcontextprotocol/server-filesystem", "/路徑一", "/路徑二"]。

MCP Server 是否需要付費？

大部分 MCP Server 都是開源免費的，您可以直接從 npm 官方套件庫安裝使用。
